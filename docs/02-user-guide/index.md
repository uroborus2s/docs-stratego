# 用户指南

本目录面向两类实际使用者：

- 文档站访问者：匿名浏览公开页面，登录后访问私有页面
- 部署维护者：负责在云服务器上安装、配置、发布和回滚 `docs-stratego`

当前项目的“用户使用基础”就是部署与安装，因此本目录默认按下面顺序阅读：

1. [用户指南](user-guide.md)
2. [安装说明](installation.md)
3. [配置说明](configuration.md)
4. [使用说明](usage.md)
5. [管理员指南](admin-guide.md)

如果你是第一次部署，请直接看安装说明；完整的服务器安装、GitHub App、证书和自动发布步骤都以这份文档为准。
