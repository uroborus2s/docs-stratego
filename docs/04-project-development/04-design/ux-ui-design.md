# UI 与信息架构

## 浏览体验

- 全部页面都显示在同一套左侧导航里
- 公开页面匿名可直接读
- 私有页面链接显示“锁定”标记
- 匿名用户点击私有页面时，先同步拉起独立的小窗登录框
- 登录链路在小窗内完成，通过同源桥接页用 `postMessage` 通知主页面
- 浏览器如果不支持严格的小窗形态，允许退化为新标签页，但主页面不主动跳走
- 用户关闭登录小窗后，应留在当前公开页面继续浏览
- 登录成功后回到原页面

## 信息架构原则

- 目录顺序由源仓 `index.md` 控制
- 页面标题直接来自 Markdown 一级标题
- 私有页面最好收敛到逻辑明确的目录中，避免资源权限混杂
